The <a href="vim.profile.host.HostMemoryProfile.md">HostMemoryProfile</a> data object represents  memory configuration for the host.  This may not be valid all versions of the host.  <p>  Use the <a href="vim.profile.ApplyProfile.md#policy">policy</a> list for access to configuration data  for the host memory profile. Use the <a href="vim.profile.ApplyProfile.md#property">property</a> list  for access to subprofile configuration data, if any.
